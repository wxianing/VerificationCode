# VerificationCode
短信验证码输入框，简单例子！

### 如何使用？

#### Step 1. Add the JitPack repository to your build file 
	```allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  #### Step 2. Add the dependency
  
  ```
  dependencies {
	        implementation 'com.github.wxianing:VerificationCode:1.0'
	}
  ```
